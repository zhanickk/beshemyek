## План: автоопределение языка для ответов бота

Сейчас системный промпт AI задан на английском, и иcебрейкеры в библиотеке — на английском. Обновлю поведение так, чтобы бот отвечал на языке пользователя.

### Что изменю

1. **AI-ответы на @mentions** (`src/routes/api/public/telegram/webhook.ts`)
   - Обновлю system prompt: «Ты добрый, дружелюбный помощник в групповом чате. ВСЕГДА отвечай на том же языке, на котором написано последнее сообщение пользователя (русский → русский, английский → английский и т.д.). Тон: тёплый, краткий, поддерживающий.»
   - Это покрывает RU/EN/любой другой язык без явных настроек.

2. **Команды бота** (`/icebreaker`, `/trivia`, `/poll`, `/start`, `/help`)
   - Сделаю двуязычные ответы: определяю язык по `from.language_code` пользователя Telegram (если `ru` → русский, иначе английский).
   - Тексты подсказок, заголовки игр и сообщения об ошибках — в обоих вариантах.

3. **Библиотека промптов** (таблица `prompts`)
   - Добавлю колонку `language` (`'en' | 'ru'`, по умолчанию `'en'`).
   - Засею 20 русских иcебрейкеров (аналоги существующих).
   - При выборе ежедневного промпта в `prompt-tick` и `/icebreaker` буду фильтровать по языку чата.

4. **Настройка языка чата** (`bot_settings`)
   - Добавлю поле `language` (`'auto' | 'en' | 'ru'`, по умолчанию `'auto'`).
   - В режиме `auto` — определяется по большинству недавних сообщений или по `language_code` отправителя команды.
   - В дашборде на странице **Chats** добавлю селектор языка для каждого чата.

5. **Дашборд** (`src/routes/_authenticated/_app/prompts.tsx`)
   - Добавлю переключатель языка при создании/редактировании промпта и фильтр в списке (RU/EN/Все).

### Технические детали

- Миграция: `ALTER TABLE prompts ADD COLUMN language text DEFAULT 'en'`, `ALTER TABLE bot_settings ADD COLUMN language text DEFAULT 'auto'`, плюс INSERT 20 русских промптов.
- Хелпер `detectLanguage(message)` в `src/lib/telegram.server.ts`: простая эвристика — наличие кириллицы → `ru`, иначе fallback на `from.language_code` или `'en'`.
- Server function `updateChatLanguage` в `src/lib/bot.functions.ts`.

### Что НЕ меняется

- UI дашборда остаётся на английском (это админка для тебя).
- Существующие английские промпты сохраняются.
- Структура webhook/cron не меняется.
